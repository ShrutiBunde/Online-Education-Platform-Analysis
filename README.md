# Online Education Platform Analysis

## Project Overview
This project analyzes student engagement, course completion, and learning patterns
on an online education platform using SQL, Pandas, and Power BI.

## Tools & Technologies
- Python (Pandas)
- MySQL
- Power BI
- Kaggle Dataset (Online Education)

## Project Structure
- data/  
  Contains the final cleaned dataset used for analysis.

- sql/  
  Contains SQL queries used to analyze enrollments, engagement, and completion rates.

- pandas/  
  Jupyter Notebook for data cleaning, feature engineering, and analysis.

- powerbi/  
  Power BI dashboard for visualizing engagement and course performance.

## Key Analysis Performed
- Total enrollments per course
- Student engagement levels
- Course completion rates
- Dropout identification
- Course ranking based on engagement

## Note
Some tasks such as satisfaction ratings and assignment submission analysis
were not performed due to unavailability of relevant data columns in the dataset.
